# Agentic Workflow and Prompt Engineering

This document describes the AI agents involved in the project.

## Workflow
1. **Intent Recognition**: The system analyzes the user's query to determine the intent.
2. **Query Generation**: Based on the intent, a ShopifyQL or GraphQL query is generated.
3. **Execution**: The query is executed against the Shopify API.
4. **Explanation**: The results are interpreted and explained to the user.
